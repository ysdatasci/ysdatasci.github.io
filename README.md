# 黄彦松个人主页

这是黄彦松的个人主页项目，用于部署到 EdgeOne Pages 或其他静态网站托管平台。

网站内容包括：

- 个人基本介绍
- 教育背景与成长轨迹
- 实习经历与关键成果
- 项目经历与开源项目
- 技术能力地图
- 荣誉奖项与技能证书
- 联系方式与个人履历 PDF 下载

## 项目信息

- 项目名称：`huang-yansong-homepage`
- 项目类型：静态个人主页
- 部署平台：EdgeOne Pages
- 构建命令：无需构建
- 输出目录：项目根目录

## 文件说明

- `index.html`：页面内容结构
- `styles.css`：页面视觉样式与响应式布局
- `script.js`：主题切换、滚动动效与项目筛选交互
- `assets/`：网站视觉资源与个人履历 PDF

## EdgeOne 部署建议

如果使用 EdgeOne Pages 连接代码仓库：

- Framework preset：Other
- Build command：留空
- Output directory：`.` 或留空
- Install command：留空

如果使用直接上传方式，将项目根目录中的所有文件上传即可。
