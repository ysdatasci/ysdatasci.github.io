# 黄彦松个人主页

这是用于部署在 GitHub Pages 上的个人主页项目，访问地址：

https://ysdatasci.github.io/

网站内容包括：

- 个人基本介绍
- 教育背景与成长轨迹
- 实习经历与关键成果
- 项目经历与开源项目
- 技术能力地图
- 荣誉奖项与技能证书
- 联系方式与个人履历 PDF 下载

该项目为静态网页项目，主要文件包括：

- `index.html`：页面内容结构
- `styles.css`：页面视觉样式与响应式布局
- `script.js`：主题切换、滚动动效与项目筛选交互
- `assets/`：网站视觉资源与个人履历 PDF
